# 项目简介

## 核心功能
- 基于YOLOv8的苹果状况检测，支持4类分类：good apple、soso apple、rotten apple、disease apple
- 提供5000张标注数据集
- 包含完整源码、训练好的模型及训练结果、项目运行教程
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 提供UI界面，可实时显示目标位置、目标总数、置信度等信息
- 支持自定义置信度和IOU阈值
- 支持选择显示标签和原图
- 支持选择检测模型
- 支持批量检测并查看所有结果
- 支持检测结果保存

## 适用场景
- 苹果质量分级检测
- 苹果病害识别
- 农产品自动化检测
- 视频监控中的苹果状况分析
- 批量图片处理与分类

## 技术特点
- 采用Python 3.9开发，依赖opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 提供Pycharm+Anaconda或VSCode+Anaconda开发环境
- 使用YOLOv8模型进行目标检测，具有较高准确率
- 提供图形用户界面，操作简便
- 支持多种输入格式，适应不同应用场景
